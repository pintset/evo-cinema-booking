# Cinema Booking
Evolution Gaming Scala Bootcamp Graduation Project

### Environment

The following tools are required:

- `sbt`
- `dotnet`
- `npm`

### Install

In the project's root folder, execute:

- `npm install`
- `npm run build`

### Run

In the project's root folder, execute:

- `sbt run`

Open the browser at `localhost:5000`
